# Flight Delays
> Flight Delays is a data science project where I use a basic linear regression model to predict the target variable arrival_delays from the flights dataset.

### Deployment
 - Clone repository
 - ```docker build -t flights .```
 - ```docker compose up```

#### Technologies
 - Scikit-learn
 - Pandas
 - FastAPI
 - Docker
 - NBConvert
 - Python
